# 🚀 Space Mission Management API

A Django REST Framework based backend API to manage space missions with JWT authentication.

## 🔧 Tech Stack
- Python
- Django
- Django REST Framework
- JWT Authentication (SimpleJWT)
- SQLite / PostgreSQL

## ✨ Features
- User authentication using JWT
- Secure API endpoints
- CRUD operations for missions
- Pagination and filtering
- RESTful architecture

## 🔐 Authentication
Uses JWT tokens for secure access:
- `/api/token/`
- `/api/token/refresh/`
